# Contributing

Thanks for your interest in JobFit Resume AI.

This project is currently a product prototype. Contributions are welcome around:

- UI and interaction improvements
- Resume template design
- ATS-friendly resume rules
- JD parsing prompts
- Evidence-chain design
- Interview review and STAR story workflows
- Next.js MVP implementation

## Suggested Contribution Areas

1. Turn the static prototype into a Next.js app.
2. Implement structured profile CRUD.
3. Add real JD parsing through AI structured outputs.
4. Build the resume editor.
5. Implement HTML-to-PDF export.
6. Improve Big Tech mode and STAR interview flows.

## Principles

- Do not encourage fabricated resume content.
- Keep generated resume claims traceable to user-provided evidence.
- Optimize for real job-search workflows, not just one-time resume generation.
- Keep the UI calm, practical, and easy to scan.
