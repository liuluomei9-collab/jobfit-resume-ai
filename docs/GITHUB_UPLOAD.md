# GitHub Upload Guide

## Option 1: Upload With Git Commands

Create an empty GitHub repository first, then run:

```powershell
git remote add origin https://github.com/YOUR_USERNAME/jobfit-resume-ai.git
git branch -M main
git push -u origin main
```

## Option 2: Upload Through GitHub Website

1. Create a new repository on GitHub.
2. Upload the project files from this folder.
3. Do not upload `.zip` archives unless you specifically want release packages.

## Recommended Repository Settings

- Repository name: `jobfit-resume-ai`
- Description: `AI-powered job application workspace for JD-based resume tailoring, application tracking, and interview review feedback.`
- Visibility: Public if you want people to discover it.
- Topics:
  - `resume`
  - `ai`
  - `job-search`
  - `ats`
  - `career`
  - `interview-prep`
  - `application-tracker`
