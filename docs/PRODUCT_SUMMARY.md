# Product Summary

JobFit Resume AI is an AI job-search workspace.

It combines:

- Career profile library
- JD analysis
- Resume tailoring
- Evidence-chain validation
- PDF export
- Application CRM
- Interview review
- Big Tech / STAR preparation

## Key Differentiator

Most resume tools stop after generating a resume. JobFit is designed to improve after each job application and interview.

## MVP Boundary

The first real MVP should include:

- Profile CRUD
- Resume import
- JD parsing
- Resume generation and editing
- PDF export
- Application tracking
- Interview review

## Long-Term Direction

The product can evolve into a complete AI career operating system:

- Personal website generation
- Batch JD analysis
- Big Tech mode
- STAR story library
- Job-search analytics
- Cover letter and outreach generation
